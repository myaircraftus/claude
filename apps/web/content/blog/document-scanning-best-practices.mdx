# Scanning Paper Logbooks

If you are scanning your own logbooks for upload, a few settings make the difference between pristine data extraction and a page-by-page manual review.

## Resolution

Scan at **300 DPI minimum**, 600 DPI if you have handwritten entries in pencil or faded ink. Lower resolutions lose the detail the OCR pipeline needs.

## Color vs. grayscale

Color is better. OCR can use red ink from a signature or a stamp as a cue that grayscale destroys.

## Flatbed, not feeder

Sheet feeders pull pages at an angle. Use a flatbed scanner (or a phone scanner app with edge-detection) for the best results. If you must use a feeder, choose one with a straight paper path.

## File format

PDF is ideal. JPEG is acceptable. Do not use compressed formats for handwritten pages — the compression artifacts will degrade OCR.
